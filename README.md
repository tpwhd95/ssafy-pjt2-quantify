## - 프로젝트 사용방법

### :floppy_disk: Install

> Using npm

**Backend**

```sh
cd backend
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py initialize
python manage.py runserver
```

**Frontend**

```sh
cd frontend
npm install
npm run serve
```
